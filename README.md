# CareerPath AI 🎓

## HOW TO RUN (VS Code)

### Step 1 - Backend
```
cd backend
npm install
node server.js
```
Backend runs on: http://localhost:5000

### Step 2 - Frontend (new terminal)
```
cd frontend
npm install
npm start
```
App opens on: http://localhost:3000

## Features
- Splash Screen with animation
- Career After 10th (Streams + Jobs)
- Career After 12th (MPC, BiPC, MEC, CEC, Arts)
- Career After Graduation (Jobs, Higher Study, Study Abroad)
- Search System
- Bottom Navigation
